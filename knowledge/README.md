# Cross-problem knowledge

This directory contains results and methods reusable across dossiers. Promotion
here does not strengthen a claim: every note must retain its proof or source,
hypotheses, provenance, and known limitations.

- [definitions.md](definitions.md) — shared terminology and conventions.
- [lemmas.md](lemmas.md) — reusable proved or imported lemmas.
- [methods.md](methods.md) — successful proof and search patterns.
- [counterexamples.md](counterexamples.md) — reusable constructions and warnings.
- [bibliography.md](bibliography.md) — sources inspected across problems.

Assign `K-###` IDs sequentially. Link back to the originating problem and record
later corrections without erasing history.
