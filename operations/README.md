# Repository operations

`O###-*.md` records material repository work that is outside mathematical
investigation: process changes, tooling, dashboards, build systems, formatting,
and infrastructure. Create one with:

```sh
python3 scripts/proofctl.py operation "Short operational title"
```

Operational records do not change a problem's mathematical status, claim ledger,
obligations, proof, reviews, or research-session estimate. Mixed tasks must split
their mathematical and operational portions into the appropriate records.
