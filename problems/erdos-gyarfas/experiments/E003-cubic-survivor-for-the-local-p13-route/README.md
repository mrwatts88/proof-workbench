# E003 — Cubic survivor for the local P13 route

- Date: 2026-07-23
- Problem: `P-002`
- Evidence class: exploratory

## Question

Can a cubic graph of the smallest possible order containing an induced
\(P_{13}\) avoid both \(C_4\) and \(C_8\)?

## Logical scope

An induced \(P_{13}\) in a cubic graph has \(2+11+2=15\) incident edges from
the path to its complement: two at each endpoint and one at each internal path
vertex. Because an outside vertex has degree at most \(3\), at least five
outside vertices are required, so the smallest possible order is \(18\). At
order \(18\), the five outside vertices have total degree \(15\), all consumed
by the path incidences; hence they are necessarily independent and each has
three path neighbors.

The search is therefore exhaustive for all order-\(18\) cubic graphs containing
a specified induced \(P_{13}\). It does not cover larger cubic attachments:
additional outside vertices may have edges among themselves. A negative result
rules out only the smallest order and does not establish a general
cycle-forcing lemma.

The result is recorded as tested evidence `C010`. The separate mathematical
limitation `A004/L007` uses large-girth cubic graphs and does not depend on this
computation.

## Environment

- Tool and version: CPython 3.14.2
- Dependencies: Python standard library only
- Exact arithmetic / floating point: integer bit masks and exact counts
- Random seed: none; deterministic exhaustive backtracking

## Inputs and search space

- Vertices \(0,\ldots,12\) induce the path
  \(0-1-\cdots-12\).
- Vertices \(13,\ldots,17\) form an independent set and each has exactly three
  neighbors on the path.
- Each internal path vertex has one outside neighbor and each endpoint has two,
  making the completed graph cubic.
- The five outside vertices are interchangeable. Their path-neighbor triples
  are generated in nondecreasing lexicographic order, which enumerates every
  multiset of five triples exactly once.
- A branch is rejected as soon as it creates a \(4\)- or \(8\)-cycle. These
  properties are monotone under adding edges, so the pruning loses no survivor.
- Exact simple-cycle detection uses bit masks and depth-first search with the
  least cycle vertex fixed as root.

rejected by a \(4\)- or \(8\)-cycle.

## Interpretation

There is no cubic graph on \(18\) vertices containing an induced \(P_{13}\)
that avoids both \(C_4\) and \(C_8\). The conclusion does not extend to larger
orders.

## Independent checks

- The cycle detector is self-tested on a tree and on isolated cycle graphs of
  lengths \(4\), \(8\), and \(16\).
- The degree demand has total \(15\), exactly the capacity of five degree-\(3\)
  outside vertices; hence every graph in the stated family corresponds to five
  neighbor triples generated by the program.
- Sorting the five triples merely quotients the permutation symmetry of the
  outside vertices.
